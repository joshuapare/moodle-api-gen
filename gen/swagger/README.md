OpenAPI Definition and Swagger UI for Moodle, generated from the relevant OpenAPI spec from the [moodle-api-gen](https://www.github.com/joshuapare/moodle-api-gen) project.

## Usage

### Running the Swagger UI

You can run the Swagger UI with any simple webserver from this directory. For example, you can install `http-server` globally with brew:

```sh
brew install http-server
```

Then run the server:

```sh
http-server
``
